# Number Guessing Game 🎯

A fun and interactive **Number Guessing Game** built using **Python Tkinter**. The game challenges players to guess a randomly generated number within a limited number of attempts, with difficulty levels and a live score tracker.

---

## Features

* **Two Difficulty Levels**

  * **Easy:** Number between 1–50 with 7 attempts.
  * **Hard:** Number between 1–100 with 5 attempts.
* **Score Tracking:** Keeps track of your correct guesses.
* **Feedback System:** Hints like "Too High", "Too Low", or "Very Close!" after each guess.
* **Attempts Progress Bar:** Shows attempts left visually.
* **Endgame Panel:** Displays result and allows you to play again or quit.
* **GUI Design:** Dark mode theme with intuitive buttons and labels.


---

## Installation

1. Make sure you have **Python 3.x** installed.
2. Install Tkinter (usually comes pre-installed with Python).

   ```bash
   pip install tk
   ```
3. Clone this repository or download the files:

   ```bash
   git clone https://github.com/yourusername/number-guessing-GUI-game.git
   ```
4. Navigate to the project folder:

   ```bash
   cd number-guessing-game
   ```

---

## How to Play

1. Run the game:

   ```bash
   python number_guessing_game.py
   ```
2. Choose a difficulty level (Easy or Hard).
3. Enter your guess in the input box and click **GUESS**.
4. Follow the feedback messages:

   * Too High ⬆️
   * Too Low ⬇️
   * Very Close 🔥
5. Keep guessing until you find the correct number or run out of attempts.
6. At the end, you can choose to **Play Again** or **Quit**.
